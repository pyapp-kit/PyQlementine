## Installation

```sh
pip install PyQt6-Qlementine
# or 
pip install PySide6-Qlementine
```

## Usage

See [C++ documentation](https://oclero.github.io/qlementine) for more information.

See also [examples in this repo](./examples/).

The autogenerated type stubs are also useful for understanding the API.

## Developers

### Prerequisites

Install Qt locally (required for building the C++ bindings):

```sh
uvx --from aqtinstall aqt install-qt mac desktop 6.8.1 --outputdir Qt
```

### Build and test locally

```sh
uv sync --no-editable
uv run pytest
```

### Build wheels with cibuildwheel

```sh
QT_VERSION=6.8.1 uvx cibuildwheel --config-file pyproject.toml packages/PyQt6-Qlementine
```
